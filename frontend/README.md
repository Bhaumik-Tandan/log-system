# Frontend

React app for viewing and filtering logs.

## Setup

```bash
yarn install
yarn dev
```

## Features

- View logs in a list
- Filter by level, message, resource, time
- Real-time filtering
- Basic statistics

## Components

- `FilterBar` - Filter controls
- `LogList` - Log display
- `App` - Main component

## API Integration

Uses axios to communicate with the backend API.
